# 📄 Digital Craft | الحرفة الرقمية 🚀

هو مشروع واجهة أمامية (Front-end) متكامل لوكالة خدمات رقمية، يجمع بين الإبداع البصري والدقة البرمجية. تم بناء المشروع باستخدام تقنيات الويب الأساسية لضمان السرعة، الخفة، وتجربة مستخدم استثنائية.

---

## ✨ المميزات الرئيسية (Key Features)

* **🌓 نظام الألوان الذكي:** دعم كامل للوضع الداكن (Dark Mode) والوضع الفاتح (Light Mode) مع حفظ تفضيلات المستخدم عبر `darkMode.js`.
* **🌐 دعم اللغات (Bilingual):** نظام تبديل لغات ديناميكي (عربي/إنجليزي) عبر ملفات JavaScript المتخصصة `language.js`.
* **📱 تصميم متجاوب (Responsive Design):** متوافق تماماً مع جميع مقاسات الشاشات والهواتف الذكية.
* **🔒 حماية الصفحات (Auth Guard):** نظام برمجي لتقييد الوصول إلى ميزات معينة (مثل طلب الخدمات) إلا في حال تسجيل الدخول.
* **🎨 واجهات عصرية:** يعتمد على خطوط **Work Sans** من Google Fonts ومكتبة **Font Awesome** للأيقونات.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لهيكلة المحتوى بشكل دلالي (Semantic UI).
* **CSS3:** للتنسيق باستخدام Flexbox و Grid مع استخدام `normalize.css` لضمان اتساق المظهر.
* **Vanilla JavaScript:** لإدارة المنطق البرمجي الكامل (تبديل اللغات، الوضع الداكن، النوافذ المنبثقة).

---

## 💼 الخدمات المعروضة
* **تطوير المواقع:** إنشاء مواقع إلكترونية تفاعلية ومرنة ومتوافقة مع جميع الأجهزة.
* **تطوير تطبيقات الجوال:** تصميم وبرمجة تطبيقات الأندرويد والـ iOS بأداء ممتاز.
* **تصميم الهوية البصرية:** ابتكار شعارات وهويات بصرية تميز العلامات التجارية.
* **تصميم المطبوعات:** تصميم أغلفة احترافية للطباعة أو النشر الإلكتروني.

## 🚀 التشغيل والتطوير
1. **التحميل:** قم بتحميل المشروع (Clone/Download).
2. **التشغيل:** افتح ملف `index.html` في أي متصفح حديث.
3. **ملاحظة تقنية:** المشروع حالياً واجهة أمامية (Front-end UI) جاهزة للربط مع أي واجهة خلفية (Back-end) أو قاعدة بيانات لجعل ميزات تسجيل الدخول وإرسال الطلبات تعمل بشكل ديناميكي حقيقي.
